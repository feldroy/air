# Air Reference

Here is the Air reference documentation. It explains how to do things, as well as providing reference details for nearly every object in the Air code.

- [Applications](http://feldroy.github.io/air/api/applications/index.md) - The app instantiator for Air
- [Background Tasks](http://feldroy.github.io/air/api/background/index.md) - Background tasks for Air
- [Exception Handlers](http://feldroy.github.io/air/api/exception_handlers/index.md) - Exceptions are returned to the user, specifically 404 and 500
- [Exceptions](http://feldroy.github.io/air/api/exceptions/index.md) - Sometimes it's good to know exactly what is breaking
- [Forms](http://feldroy.github.io/air/api/forms/index.md) - Receive and validate data from users on web pages
- [Layouts](http://feldroy.github.io/air/api/layouts/index.md) - Utilities for building layout functions and two example layouts for css microframeworks (mvcss and picocss)
- [Middleware](http://feldroy.github.io/air/api/middleware/index.md) - Middleware for Air
- [Requests](http://feldroy.github.io/air/api/requests/index.md) - HTMX utility function that can be used with dependency injection
- [Responses](http://feldroy.github.io/air/api/responses/index.md) - AirResponse for normal responses and SSEResponse for Server Sent Events
- [Routing](http://feldroy.github.io/air/api/routing/index.md) - For compositing multiple apps inside each other
- [SVG](http://feldroy.github.io/air/api/svg/index.md) - Reference for the entire SVG tags specification, all of which are supported by Air
- [Tags](http://feldroy.github.io/air/api/tags/index.md) - Reference for the entire HTML tags specification, all of which are supported by Air
- [Templating](http://feldroy.github.io/air/api/templating/index.md) - Describes Jinja and Air Tag renderers for use in both projects and third-party installable packages
- [Utils](http://feldroy.github.io/air/api/utils/index.md) - Utilities that don't fall into one of the above categories
